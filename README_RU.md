# Цифровой двойник энергосистемы ЦОД

## Описание проекта

Данный проект представляет собой симуляционную модель цифрового двойника энергосистемы дата-центра.

Система генерирует реалистичный набор телеметрических данных для:
- моделирования нагрузки ЦОД;
- анализа энергопотребления;
- обучения ML-моделей;
- прогнозирования аварий;
- оптимизации работы энергосистемы.

Модель имитирует работу:
- вычислительной нагрузки;
- графических процессоров;
- аккумуляторной системы;
- системы охлаждения;
- температурных режимов;
- прогнозной подсистемы;
- аварийных сценариев.

---

# Основные возможности

## Энергосистема
- динамическое изменение нагрузки;
- инерция изменения мощности ГПУ;
- логика заряда и разряда аккумуляторов;
- обработка пиковых нагрузок;
- аварийные режимы работы.

## Система охлаждения
- моделирование внешней температуры;
- моделирование температуры серверной;
- расчёт мощности охлаждения;
- расчёт коэффициента PUE.

## Аварийные события

Поддерживаются следующие типы аварий:
- отказ ГПУ;
- отказ системы охлаждения;
- отказ аккумуляторной системы;
- нестабильность электросети;
- отказ датчиков.

## ML-признаки

В датасете присутствуют:
- скользящие средние;
- лаговые признаки;
- ошибки прогнозирования;
- дефицит мощности;
- температурные показатели;
- целевые признаки для ML.

---

# Характеристики датасета

- 40 000 строк телеметрии;
- период моделирования — 60 дней;
- реалистичное поведение временных рядов;
- экспорт в CSV и XLSX.

---

# Целевые признаки для ML

В датасете содержатся:
- `target_load_1h` — прогноз нагрузки через 1 час;
- `target_peak_1h` — прогноз пикового события через 1 час.
